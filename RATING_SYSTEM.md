# ⭐ Система баллов и рейтинга ГрузПоток

## 📊 Как работает система

### Начисление баллов:

| Действие | Баллы | Описание |
|----------|-------|----------|
| ✅ Успешная сделка | +10 | Когда ставка принята и заявка закрыта |
| ✅ Завершённая заявка | +5 | Когда заявка успешно выполнена |
| ➕ Создание заявки | +2 | За каждую созданную заявку |
| 💰 Создание ставки | +2 | За каждую ставку на заявку |
| ✅ Верификация аккаунта | +50 | Бонус за верификацию |

### Штрафы:

| Действие | Баллы | Описание |
|----------|-------|----------|
| ⚠️ Жалоба | -20 | За каждую полученную жалобу |
| ⚠️ Спор | -50 | За каждый спор по сделке |
| 🚨 Мошенничество | -100 | За подтверждённое мошенничество |

### Уровни доверия:

| Уровень | Баллы | Рейтинг | Описание |
|---------|-------|---------|----------|
| 🆕 Новый | 0+ | 0.0+ | Только зарегистрировался |
| ✅ Доверенный | 50+ | 3.5+ | Проверенный пользователь |
| ⭐ Верифицированный | 200+ | 4.0+ | Высокий уровень доверия |
| 💎 Премиум | 500+ | 4.5+ | Максимальный уровень |

## 📈 Расчёт рейтинга

Рейтинг рассчитывается автоматически на основе:

1. **Базовый рейтинг:** 5.0
2. **Штрафы:**
   - -0.1 за каждую жалобу
   - -0.2 за каждый спор
3. **Бонусы:**
   - +0.05 за каждую успешную сделку (макс +1.0)
   - +0.5 если аккаунт верифицирован
   - +0.5 за время на платформе (макс за год)

**Итоговый рейтинг:** от 0.0 до 5.0

## 🔄 Автоматическое начисление

Баллы начисляются автоматически при:

- ✅ Создании заявки
- ✅ Создании ставки
- ✅ Принятии ставки (успешная сделка)
- ✅ Завершении заявки

## 📱 API Эндпоинты

### Получить свою статистику:
```bash
GET /rating/my-stats
Authorization: Bearer <token>
```

### Получить статистику пользователя:
```bash
GET /rating/stats/{user_id}
```

### Топ пользователей:
```bash
GET /rating/leaderboard?limit=10&role=carrier
```

### Верификация пользователя (админ):
```bash
POST /rating/verify/{user_id}
Authorization: Bearer <admin_token>
```

## 💡 Примеры использования

### Проверка своего рейтинга:
```javascript
const response = await fetch('/rating/my-stats', {
    headers: { 'Authorization': `Bearer ${token}` }
});
const stats = await response.json();
console.log(`Баллы: ${stats.points}, Рейтинг: ${stats.rating}`);
```

### Просмотр топа:
```javascript
const response = await fetch('/rating/leaderboard?limit=10');
const leaderboard = await response.json();
```

## 🎯 Цели системы

1. **Мотивация** - пользователи получают баллы за активность
2. **Доверие** - рейтинг показывает надёжность пользователя
3. **Защита** - низкий рейтинг предупреждает о рисках
4. **Конкуренция** - топ пользователей мотивирует к активности
