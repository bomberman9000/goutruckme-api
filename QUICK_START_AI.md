## 🚀 Для включения LLM (опционально)

Если хотите использовать настоящий ИИ (GPT), выполните 3 шага:

### 1. Создайте файл `.env`

```bash
cd /path/to/gruzpotok
touch .env
```

### 2. Добавьте настройки

Откройте `.env` и добавьте:

```bash
# Включить LLM
AI_USE_LLM=true

# OpenAI (получить ключ: https://platform.openai.com/api-keys)
OPENAI_API_KEY=sk-your-key-here
```

### 3. Перезапустите сервер

```bash
# Остановите текущий сервер (Ctrl+C)
# Запустите снова
bash start_server.sh
# или напрямую:
python3 -m uvicorn app.api.main:app --host 0.0.0.0 --port 8080 --reload
```
