## 🧪 Как проверить вручную

### Шаг 1: Открыть страницу "Клиенты"
1. Откройте http://localhost:8080
2. Нажмите "Клиенты" в навигации
3. **Ожидаемый результат:** Откроется страница с таблицей клиентов (пока пустая)

### Шаг 2: Импорт клиентов
1. Нажмите "📥 Импорт"
2. Вставьте данные из таблицы (например, первые несколько строк)
3. Нажмите "🔍 Распознать"
4. **Ожидаемый результат:**
   - Появится preview таблица с распознанными клиентами
   - Показывается статистика
   - Видны типы организаций, ИНН, телефоны

5. Нажмите "✅ Импортировать"
6. **Ожидаемый результат:** Клиенты добавлены, таблица обновилась

### Шаг 3: Поиск и фильтры
1. Введите поисковый запрос (название, ИНН, телефон)
2. **Ожидаемый результат:** Таблица фильтруется

3. Выберите фильтр по типу (например, "ИП")
4. **Ожидаемый результат:** Показываются только ИП

### Шаг 4: Добавление клиента вручную
1. Нажмите "➕ Добавить"
2. Заполните форму:
   - Тип: ООО
   - Название: ООО "Тест"
   - ИНН: 1234567890
   - Телефон: +7 999 123-45-67
   - Адрес: г. Москва, ул. Тестовая, д. 1
3. Нажмите "Сохранить"
4. **Ожидаемый результат:** Клиент добавлен в таблицу

### Шаг 5: Редактирование клиента
1. Нажмите ✏️ на клиенте в таблице
2. Измените данные
3. Нажмите "Сохранить"
4. **Ожидаемый результат:** Данные обновлены

### Шаг 6: Интеграция в сделку
1. Откройте "Сделки"
2. Откройте любую сделку
3. Найдите блок "Клиент (Контрагент)"
4. Введите название или ИНН клиента в поле поиска
5. **Ожидаемый результат:** Появится dropdown с результатами поиска

6. Выберите клиента из списка
7. **Ожидаемый результат:** Клиент выбран и отображается в блоке

### Шаг 7: Генерация PDF с клиентом
1. В сделке с выбранным клиентом нажмите "Сформировать" на документе
2. Заполните параметры документа
3. Сгенерируйте PDF
4. **Ожидаемый результат:**
   - В PDF есть блок "КОНТРАГЕНТ (КЛИЕНТ)" или "ПОКУПАТЕЛЬ (КОНТРАГЕНТ)"
   - Показываются все данные клиента из базы
   - Если клиент не выбран → показывается "Клиент не выбран"

### Шаг 8: Проверка сохранения
1. Обновите страницу (F5)
2. **Ожидаемый результат:**
   - Клиенты сохраняются
   - Выбранные клиенты в сделках сохраняются

## 🔧 Технические детали

### Хранение данных
- **Клиенты:** `gotruckme_clients_v1`
- **Сделки:** `gotruckme_deals_v1` (с полем `counterpartyId`)

### Парсер импорта
- **Разделители:** табуляция (`\t`) или 2+ пробела
- **ИНН:** извлекается regex `/\b\d{10,12}\b/` из всех полей
- **Тип организации:** определяется по ключевым словам в названии
- **Телефон:** нормализуется до формата `+7 XXX XXX-XX-XX`

### Интеграция
- Клиент выбирается в DealDetails через поиск
- Сохраняется `counterpartyId` в Deal
- В PDF подставляются все данные клиента из базы
