# ⚠️ Система претензий и форум ГрузПоток

## 📋 Система претензий

### Как работает:

1. **Подача претензии:**
   - Укажите пользователя, на которого жалуетесь
   - Опишите проблему
   - Выберите тип претензии
   - Приложите доказательства (ссылки на фото, документы)

2. **Автоматические действия:**
   - Претензия отправляется на рассмотрение администратору
   - Пользователю автоматически снимается -20 баллов
   - Претензия сохраняется в истории

3. **Рассмотрение:**
   - Администратор рассматривает претензию
   - Может подтвердить, отклонить или запросить дополнительную информацию
   - При подтверждении дополнительно снимаются баллы

### Типы претензий:

- **Общая** - общие проблемы
- **Мошенничество** - обман, невыполнение обязательств
- **Задержка** - задержка доставки
- **Повреждение груза** - порча товара
- **Проблемы с оплатой** - неоплата, задержка платежа

## 💬 Форум недобросовестных перевозчиков

### Возможности:

1. **Создание постов:**
   - Предупреждения о недобросовестных перевозчиках
   - Отзывы о работе
   - Жалобы
   - Обсуждения

2. **Поиск:**
   - По названию компании
   - По телефону
   - По содержанию

3. **Модерация:**
   - Все посты проверяются модератором
   - Верифицированные посты помечаются галочкой
   - Администратор может закрепить важные посты

4. **Взаимодействие:**
   - Лайки/дизлайки
   - Комментарии
   - Просмотры

## 🔗 API Эндпоинты

### Претензии:

```bash
# Создать претензию
POST /complaints/create
Authorization: Bearer <token>
{
  "defendant_id": 123,
  "load_id": 456,
  "title": "Не выполнил перевозку",
  "description": "Подробное описание...",
  "complaint_type": "fraud",
  "evidence": "ссылки на фото"
}

# Мои претензии
GET /complaints/my-complaints
Authorization: Bearer <token>

# Претензии против меня
GET /complaints/against-me
Authorization: Bearer <token>

# Претензии пользователя (публично)
GET /complaints/user/{user_id}
```

### Форум:

```bash
# Создать пост
POST /forum/create
Authorization: Bearer <token>
{
  "title": "Внимание! Недобросовестный перевозчик",
  "content": "Описание ситуации...",
  "post_type": "warning",
  "target_company": "ООО Компания",
  "target_phone": "+7 999 123-45-67"
}

# Список постов
GET /forum/list?post_type=warning&search=компания

# Просмотр поста
GET /forum/{post_id}

# Добавить комментарий
POST /forum/{post_id}/comment

# Лайк/дизлайк
POST /forum/{post_id}/like
POST /forum/{post_id}/dislike
```

## 🎯 Интеграция с системой баллов

При подаче претензии:
- Автоматически снимается **-20 баллов** у пользователя
- При подтверждении администратором - дополнительно **-20 баллов**
- Увеличивается счётчик `complaints` в профиле

## 📱 Использование в интерфейсе

1. **Подать претензию:**
   - Перейдите в раздел "Претензии"
   - Нажмите "Подать претензию"
   - Или используйте кнопку "Пожаловаться" в карточке заявки

2. **Создать пост на форуме:**
   - Перейдите в раздел "Форум"
   - Нажмите "Создать пост"
   - Заполните информацию о недобросовестном перевозчике

3. **Просмотр:**
   - Все претензии и посты доступны в соответствующих разделах
   - Можно фильтровать и искать
