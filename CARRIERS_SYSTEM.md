## 🧪 Как проверить вручную

### Шаг 1: Открыть страницу "Перевозчики"
1. Откройте http://localhost:8080
2. Нажмите "Перевозчики" в навигации
3. **Ожидаемый результат:** Откроется страница с таблицей перевозчиков (пока пустая)

### Шаг 2: Импорт перевозчиков
1. Нажмите "📥 Импорт"
2. Вставьте данные перевозчиков (таблицу с табами или пробелами)
3. Нажмите "🔍 Распознать"
4. **Ожидаемый результат:**
   - Появится preview таблица с распознанными перевозчиками
   - Показывается статистика
   - Видны типы организаций, ИНН/паспорт, телефоны, ТС

5. Нажмите "✅ Импортировать"
6. **Ожидаемый результат:** Перевозчики добавлены, таблица обновилась

### Шаг 3: Поиск и фильтры
1. Введите поисковый запрос (название, ИНН, телефон)
2. **Ожидаемый результат:** Таблица фильтруется

3. Выберите фильтр по типу (например, "ИП")
4. **Ожидаемый результат:** Показываются только ИП

### Шаг 4: Добавление перевозчика вручную
1. Нажмите "➕ Добавить"
2. Заполните форму:
   - Тип: Физ. лицо
   - ФИО: Иванов Иван Иванович
   - Телефон: +7 999 123-45-67
   - ТС: Газель, 5т
   - Паспорт: 4511 299531 выдан...
3. Нажмите "Сохранить"
4. **Ожидаемый результат:** Перевозчик добавлен в таблицу

### Шаг 5: Редактирование перевозчика
1. Нажмите ✏️ на перевозчике в таблице
2. Измените данные
3. Нажмите "Сохранить"
4. **Ожидаемый результат:** Данные обновлены

### Шаг 6: Интеграция в сделку
1. Откройте "Грузы"
2. Нажмите "Взять" на любом грузе
3. В модалке найдите блок "Перевозчик"
4. Введите название или ИНН перевозчика в поле поиска
5. **Ожидаемый результат:** Появится dropdown с результатами поиска

6. Выберите перевозчика из списка
7. **Ожидаемый результат:** Перевозчик выбран и отображается в блоке

8. Нажмите "Подтвердить"
9. **Ожидаемый результат:** Сделка создана с выбранным перевозчиком

### Шаг 7: Генерация PDF с перевозчиком
1. В сделке с выбранным перевозчиком нажмите "Сформировать" на документе
2. Заполните параметры документа
3. Сгенерируйте PDF
4. **Ожидаемый результат:**
   - В PDF есть блок "ПЕРЕВОЗЧИК" с данными из базы
   - Показываются все данные перевозчика (ИНН, телефон, адрес, реквизиты, ТС)
   - Если перевозчик не выбран → используется текущий пользователь

### Шаг 8: Очистка базы
1. На странице "Перевозчики" нажмите "🗑️ Очистить базу"
2. Подтвердите удаление
3. **Ожидаемый результат:** Все перевозчики удалены, таблица пустая

### Шаг 9: Проверка сохранения
1. Обновите страницу (F5)
2. **Ожидаемый результат:**
   - Перевозчики сохраняются
   - Выбранные перевозчики в сделках сохраняются

## 🔧 Технические детали

### Хранение данных
- **Перевозчики:** `gotruckme_carriers_v1`
- **Сделки:** `gotruckme_deals_v1` (с полем `carrierId`)

### Парсер импорта
- **Разделители:** табуляция (`\t`) или 2+ пробела
- **ИНН:** извлекается regex `/\b\d{10,12}\b/` из всех полей
- **Телефон:** извлекается regex'ом с поддержкой форматов:
  - `+7 (999) 123-45-67`
  - `8 (999) 123-45-67`
  - `79991234567`
  - `9991234567`
- **Тип организации:** определяется по ключевым словам в названии
- **Обработка пустых полей:** не падает, если нет ИНН/паспорта/ТС

### Интеграция
- Перевозчик выбирается в модалке "Взять груз" через поиск
- Сохраняется `carrierId` в Deal
- В PDF подставляются все данные перевозчика из базы
- Если перевозчик не выбран → используется текущий пользователь (обратная совместимость)
