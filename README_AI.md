## 🚀 Быстрый старт

### Вариант 1: Оставить как есть (рекомендуется для начала)

**Ничего делать не нужно!** ИИ уже работает в локальном режиме.

### Вариант 2: Включить LLM (для продакшена)

1. **Создайте файл `.env`** в корне проекта:

```bash
# Включить LLM
AI_USE_LLM=true

# Выберите один из провайдеров:

# OpenAI (самый мощный)
OPENAI_API_KEY=sk-your-key-here

# ИЛИ Yandex GPT (российский)
YANDEX_GPT_KEY=your-key-here
YANDEX_FOLDER_ID=your-folder-id

# ИЛИ GigaChat (бесплатный для тестов)
GIGACHAT_KEY=your-key-here
```

2. **Перезапустите сервер**

3. **Проверьте статус:**
```bash
python3 check_ai.py
```

## 🔧 Устранение проблем

**Проблема:** "ИИ не работает"

**Решение:**
1. Проверьте: `python3 check_ai.py`
2. Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
3. Проверьте логи сервера

**Проблема:** "LLM не отвечает"

**Решение:**
1. Проверьте API ключ
2. Проверьте интернет-соединение
3. Установите `AI_FALLBACK_TO_LOCAL=true` (будет работать локальный режим)
