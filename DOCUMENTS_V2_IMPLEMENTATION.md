## 🧪 Как проверить вручную

### Шаг 1: Настройка профиля компании
1. Откройте http://localhost:8080
2. Перейдите в "Сделки"
3. Откройте любую сделку
4. Нажмите "Сформировать" на любом документе
5. **Ожидаемый результат:** Предложение настроить профиль компании
6. Заполните профиль:
   - Название организации *
   - ИНН *
   - Адрес *
   - Телефон *
   - ОГРН, Email, Директор (опционально)
7. Нажмите "Сохранить"
8. **Ожидаемый результат:** Профиль сохранён, открывается модалка параметров документа

### Шаг 2: Заполнение параметров документа
1. В модалке "Параметры документа":
   - Время погрузки: `09:00`
   - Время выгрузки: `18:00`
   - Адрес загрузки: `г. Москва, ул. Примерная, д. 1`
   - Адрес выгрузки: `г. Санкт-Петербург, ул. Примерная, д. 2`
   - Особые условия: `Хрупкий груз, осторожно`
   - Контакт на месте: `Иван Иванов, +7 999 123-45-67` (опционально)
   - Номер авто: `А123БВ777` (опционально)

2. Нажмите "Сохранить черновик"
3. **Ожидаемый результат:** Черновик сохранён, уведомление "✅ Черновик сохранён"

4. Закройте модалку и откройте снова
5. **Ожидаемый результат:** Поля автоматически заполнены из черновика

### Шаг 3: Валидация
1. Очистите обязательные поля
2. Нажмите "Сгенерировать PDF"
3. **Ожидаемый результат:** Показываются ошибки валидации

4. Заполните неверное время (например, `25:00`)
5. **Ожидаемый результат:** Ошибка "Время должно быть в формате HH:MM"

6. Заполните короткий адрес (менее 10 символов)
7. **Ожидаемый результат:** Ошибка "Адрес должен содержать минимум 10 символов"

### Шаг 4: Генерация PDF
1. Заполните все обязательные поля корректно
2. Нажмите "Сгенерировать PDF"
3. **Ожидаемый результат:**
   - PDF скачивается
   - Уведомление "✅ Документ сформирован"
   - В деталях сделки кнопка меняется на "📥 Скачать PDF"
   - Показывается дата создания

4. Откройте PDF
5. **Ожидаемый результат:**
   - Все данные автозаполнены:
     - Маршрут из сделки
     - Заказчик из профиля компании
     - Перевозчик из сделки
     - Адреса из черновика
     - Время из черновика
     - Особые условия из черновика

### Шаг 5: Проверка черновиков
1. Создайте черновик для документа
2. Закройте модалку
3. В деталях сделки проверьте отображение
4. **Ожидаемый результат:** Показывается "Черновик: [первые 40 символов адреса] ([время])"

### Шаг 6: Проверка сохранения
1. Обновите страницу (F5)
2. **Ожидаемый результат:**
   - Профиль компании сохраняется
   - Черновики сохраняются
   - Документы сохраняются

## 🔧 Технические детали

### Хранение данных
- **Профиль компании:** `gotruckme_company_profile`
- **Черновики:** `gotruckme_document_drafts_v1` (формат: `{ "dealId:docType": {...} }`)
- **Документы:** `gotruckme_documents_v1`

### Валидация
- **Время:** Регулярное выражение `/^([0-1][0-9]|2[0-3]):[0-5][0-9]$/`
- **Адреса:** Минимум 10 символов
- **Обязательные поля:** Время погрузки, время выгрузки, адреса

### Автозаполнение
- **Из сделки:** маршрут, цена, дистанция, параметры груза
- **Из профиля компании:** заказчик (название, ИНН, ОГРН, адрес, телефон)
- **Из сделки.carrier:** перевозчик
- **Из черновика:** адреса, время, особые условия, опциональные поля
